There are two types of websites

  • Dynamic website
  • Static website

DYNAMIC WEBSITE

A dynamic website is a website that displays different type of contents every time a user views it. This display changes depending on a number of factors like viewer demographics, time , locatation, language settings and so on.

STATIC WEBSITE

A static website is a website that is delivered to the user's web browser exactly as stored.
